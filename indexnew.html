Welcome

What is StatTrack?

StatTrack is a website where you can find different statistics for various things on Minecraft servers like longest bow kills etc.

How does it work?

StatTrack uses Github Pages to host a website. The website has been created by the developers that you can find on the staff page.

How often do you update StatTrack?

StatTrack keeps getting updated all the time! You can look at our progress on the updates page.

Issues

If you find an issue with the website you can create an issue or a pull request including a fix on our Github repository.
